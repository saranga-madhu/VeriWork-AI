 How to Run the Project
Follow these steps to run all services locally:

Step 1: Start MongoDB
Ensure MongoDB is running locally on port 27017.

Step 2: Start the AI Service (Python)
Ensure Python 3 is installed, then run:

VS code powershell

cd "D:\Madhu\Fraud Job Detection System\ai_module"
dir
  //cd ai_module
  pip install -r requirements.txt
  python app.py
Note: The Flask server will boot on port 5000 and automatically train its model if models/job_fraud_model.pkl is not found.

Step 3: Start the Backend (Express)
VS code powershell

  cd "D:\Madhu\Fraud Job Detection System\backend"
  //cd backend
  npm install
  node server.js
Note: The Express server will run on port 5050.

Step 4: Start the Frontend (Vite)
VS code powershell
cd "D:\Madhu\Fraud Job Detection System\frontend"
  //cd frontend
  npm install
  npm run dev
Note: Open http://localhost:5173 in your browser. Log in using seeker@test.com (password: password123) to apply to jobs, or employer@test.com (password: password123) to post openings and inspect candidate assessments.
